# Running it locally

For Windows Python3.9:

> pip install -r requirements.txt

> python manage.py migrate

> python manage.py runserver

For Mac/Linux Python3.9:

> pip install -r requirements-unix.txt

> python manage.py migrate

> python manage.py runserver
